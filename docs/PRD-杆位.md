# 📋 杆位 — 产品需求文档（PRD）

## 产品定义
**名称：** 杆位  
**定位：** F1赛车竞猜+数据小程序  
**一句话：** 猜比赛、看数据、晒积分。一个App看所有赛车。  
**目标用户：** 中国F1车迷，微信群活跃用户，硬核数据迷

---

## MVP功能清单（8个模块）

| 模块 | 功能 | 数据来源 | 优先级 |
|------|------|---------|--------|
| 🎯 竞猜 | 每站猜杆位+领奖台+最快圈，三步提交 | 自建 | P0 |
| 🏆 排名 | 赛季积分排行榜+迷你赛季（每6站） | 自建 | P0 |
| 📅 赛程 | 24站日历+赛道详情（长度/弯道/DRS） | jolpica | P0 |
| 📊 积分 | 车手积分榜+车队积分榜 | jolpica | P0 |
| 👤 我的 | 积分+准确率+预测历史+支持车队 | 自建 | P0 |
| 📤 分享 | 竞猜结果卡片可分享到微信群 | 自建 | P0 |
| 📰 快讯 | Motorsport.com RSS标题 | 自动抓取 | P0 |
| 💎 Pro | 圈速对比+历史数据+无广告 ¥18/月 | 自建 | P2 |

## 不做
- ❌ App（先小程序）
- ❌ 社区/BBS
- ❌ 新闻稿（只抓标题）
- ❌ 自建商城
- ❌ WEC/WRC/FE竞猜（无API）
- ❌ 实时数据
- ❌ 广告轰炸

---

## 技术栈
- 前端：uni-app (Vue3) → 微信小程序
- 后端：Python FastAPI + SQLite
- 数据：jolpica自部署（主）+ Fast-F1（备份）
- 资讯：Motorsport.com RSS
- 部署：阿里云ECS ¥200-400/月

## 63轮调研核心结论
- 需求真实（每日赛车800评+用户主动要登录功能+微信群有人参与竞猜）
- 竞品极弱（每日赛车死了、围场只做小组件、BoxBox不做中文）
- ASO竞争近零
- 版权安全（无F1诉讼记录）
- 天花板DAU 3-6万（中国），年入百万需全球化
- 最大风险：竞猜参与度不足——但已通过微信群验证
