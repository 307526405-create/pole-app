## 注意事项

### 1. 微信小程序主体
- 用**企业主体**注册小程序（需要营业执照）。个人主体功能受限。
- 你已有广东雅和网络科技有限公司→直接用来注册。
- 微信认证费¥300/年。

### 2. 竞猜争议处理
- "我明明选了诺里斯，系统显示我选了维斯塔潘"→这是最危险的投诉。
- **每次竞猜提交→后端记录用户ID+时间戳+完整选择内容→用户可在"我的-竞猜历史"中查看。**
- 争议出现时，以后端记录为准。给用户截图看。

### 3. 赛季冠军奖品
- 淘宝下单→填获奖者地址→直接寄。**不需要你经手。**
- 发货后给获奖者物流单号。不要让用户催"奖品呢"。
- 奖品预算：¥300-600/年。不要省。

### 4. 服务器成本曲线
- 0-1000 DAU：阿里云ECS 2核4G ¥200/月（够用）
- 1000-5000 DAU：升级4核8G ¥400/月
- 5000+ DAU：需要考虑CDN+负载均衡
- **不要一上来买大配置。** 1000 DAU都不到就花¥400/月=浪费。

### 5. 用户隐私
- 只存微信openid+竞猜记录+积分。不存手机号、地址（除非获奖）。
- 隐私政策：样板可以抄懂球帝小程序的。
